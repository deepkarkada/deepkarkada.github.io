Knowledge Graph Based Natural Language Understanding in a Rapid Spoken Dialogue Map-Game
Natural Language, Dialog and Speech Symposium (NDS2020), 2020
Deepthi Karkada, Ramesh Manuvinakurike, Maike Paetzel, Kallirroi Georgila "Knowledge Graph Based Natural Language Understanding in a Rapid Spoken Dialogue Map-Game", 2020 (Abstract) Abstract

RDG-map (Rapid dialogue games map) is a domain intended to improve the geographic literacy of the user with a fast paced conversational game between two participants - a spoken dialogue agent and a human user. Worked on several components in the design and development of the dialogue systems agent such as annotations and analysis of the collected corpus as well as the Natural language understanding (NLU) module. Worked on improving the performance of the natural language understanding module by fine-tuning a convolutional neural network-based knowledge graph model for the RDG-Map domain. Also worked on other components such as building an LSTM based dialogue act classifier and a relations classifier for knowledge graphs. Helped integrate the NLU for the development of dialogue policy.
